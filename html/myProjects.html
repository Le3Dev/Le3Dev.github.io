MEUS PROJETOS

JOGO DE XADREZ

O que me fez apaixonar pela programação foi a linguagem Java. Como meu primeiro projeto pessoal, desenvolvi um jogo de xadrez em Java, implementando todas as lógicas de um jogo tradicional.

Esse jogo me ajudou a treinar e melhorar meus conceitos em programação, utilizando a linguagem orientada a objetos. Apesar de simples contribuiu muito para o meu aprendizado por ser meu primeiro projeto.

JOGO DO MIKE

Desenvolvi meu primeiro jogo este ano, com o personagem principal inspirado no meu cachorro, Mike. O objetivo do jogo é atirar ossos para afastar os gatos, que são os inimigos.

Este projeto também foi desenvolvido em Python utilizando a biblioteca Pygame. Ele contém todas as lógicas fundamentais de um jogo, incluindo início, meio, fim e pontuação.

LOS PEDRO CUSTOM

Esse projeto foi feito com meus amigos da faculdade, com o objetivo de aprender sobre personalização de páginas e a utilização de formulários dinâmicos em sites.

Neste projeto, implementamos um formulário dinâmico que se adapta ao modelo de carro selecionado, alterando perguntas e opções. Ao final, é gerado um relatório completo com as escolhas do usuário.

CADASTRO INDÚSTRIA

Esse projeto, parte da disciplina de Raciocínio Algorítmico, foi desenvolvido em grupo, e eu fui responsável pelo back-end. Criamos um sistema de cadastro de materiais que salva as informações em um banco de dados e gera um relatório completo com os dados inseridos.

Nesse projeto, utilizamos Python como linguagem de desenvolvimento e o framework Flask, juntamente com HTML e CSS, para criar o sistema. Ele permite visualizar relatórios completos de todos os materiais cadastrados.
